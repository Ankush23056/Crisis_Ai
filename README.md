# 🚨 Crisis_Ai: Next-Gen Emergency Management & Predictive Intelligence

**Crisis_Ai** is a tactical, high-performance Emergency Management Platform designed for real-time situational awareness and proactive disaster mitigation. By utilizing the **Groq LPU (Language Processing Unit)**, the platform delivers near-instant AI analysis, transforming environmental data into life-saving operational plans.

## 🧠 AI-Driven Features

### 1. 🗺️ National Intelligence Dashboard (Proactive)
A macro-level forecasting tool that uses Groq to predict regional risks across India. It provides:
- **7-Day Probability Timelines:** Estimated dates for potential disaster occurrences.
- **High-Risk State Identification:** Strategic highlighting of vulnerable regions like Assam or Maharashtra.

### 2. 🚒 Smart Response Center (Operational)
Clicking **"Respond"** triggers a Smart Response Plan generated by Llama 3.1:
- **Immediate Action SOPs:** Step-by-step checklists for first responders.
- **Resource Allocation:** Automated calculations for required ambulances, fire units, and personnel.
- **Evacuation Routing:** AI-suggested safe paths based on real-time threat location.

### 3. 📡 Localized Risk Prediction
A granular tool where users input specific meteorological data (Temperature, Humidity, Rainfall) to calculate localized risk percentages via Transformer-based inference.

### 4. 📰 Social Radar & Crisis Chatbot
- **Social Radar:** Filters noise from social media feeds to detect genuine distress signals.
- **Health & Disaster Expert:** A persistent, empathetic chatbot for immediate safety guidance.

## ⚙️ Deployment & Setup

### 1. Clone & Install
```bash
git clone https://github.com/Ankush23056/Crisis_Ai.git
npm install
```

### 2. Environment Setup
Create a `.env.local` file in the project root and add your Groq key:
```
VITE_GROQ_API_KEY=gsk_your_key_here
```

### 3. Production Deployment
1. Push code to your GitHub repository.
2. In the **Netlify Dashboard**, add `VITE_GROQ_API_KEY` to the **Environment Variables** section.
3. Deploy with the build command:
```bash
npm run build
```
